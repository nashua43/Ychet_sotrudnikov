Постановка задачи
Разработать программу "Учёт сотрудников" для управления базой данных сотрудников предприятия с возможностью формирования трех видов отчетов с использованием сортировки слиянием.

Исходные данные:
•	База данных сотрудников в памяти программы
•	Каждая запись содержит: фамилия, имя, отчество, дата рождения (год, месяц, день), отдел, должность, оклад, стаж работы

Требуемые функции:
1.	Добавление, удаление, редактирование и просмотр записей о сотрудниках
2.	Формирование трех отчетов с использованием сортировки слиянием:
Отчет 1: Полный список всех сотрудников, отсортированный по году рождения (по убыванию) + стаж (по убыванию) + фамилия (по возрастанию)
Отчет 2: Список всех сотрудников заданного отдела, отсортированный по окладу (по убыванию) + фамилия (по возрастанию)
Отчет 3: Список всех сотрудников с окладом выше среднего по предприятию, отсортированный по должности (по возрастанию) + фамилия (по возрастанию)
3.	Защита от некорректного ввода данных
4.	Удобный пользовательский интерфейс с меню

Технические требования:
•	Использование алгоритма сортировки слиянием
•	База данных должна содержать не менее 25 записей
•	Программа должна корректно работать с различными исходными данными
